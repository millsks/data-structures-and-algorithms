# Online Resources

A curated collection of online platforms, courses, and resources for learning data structures and algorithms.

## 🎓 Interactive Learning Platforms

### LeetCode
**URL**: [leetcode.com](https://leetcode.com)  
**Type**: Problem solving platform  
**Cost**: Free (Premium: $35/month)

**Features:**
- 2500+ coding problems
- Difficulty levels: Easy, Medium, Hard
- Company-specific question lists
- Discussion forums with solutions
- Mock interviews
- Contest competitions

**Best for**: Interview preparation, practical problem-solving

**Tips:**
- Start with Easy problems
- Focus on patterns, not memorization
- Read discussion after solving
- Track progress with tags

---

## 💡 Tips for Using Online Resources

1. **Don't spread too thin**: Pick 2-3 resources and stick with them
2. **Practice consistently**: Daily practice beats marathon sessions
3. **Active learning**: Code along, don't just watch/read
4. **Track progress**: Use spreadsheets or apps to monitor
5. **Join communities**: Discord, Reddit for motivation and help
6. **Revisit concepts**: Spaced repetition improves retention
